# Restaurant Management System (Backend)

This is the **backend** for a Restaurant Management System built with **Node.js** and **Express**, using **MongoDB** as the database. Postman was used for testing the API endpoints.

## Features

- **Menu Management**: Create, read, update, and delete menu items.  
- **Order Management**: Handle customer orders and track order status.  
- **User Authentication**: Basic user registration and login (optional).  
- **Database**: MongoDB for storing menus, orders, and user data.  
- **Testing**: API endpoints tested with Postman.

## Tech Stack

- **Node.js**  
- **Express.js**  
- **MongoDB**  
- **Postman** (for testing)
